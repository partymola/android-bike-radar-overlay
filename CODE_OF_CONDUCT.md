# Code of Conduct

This is a small personal project, but everyone who takes part - filing an
issue, sending a translation, correcting the protocol, reviewing a pull
request - deserves to be treated decently.

## The short version

- Be respectful and assume good faith. People come to a cycling-safety app
  from very different backgrounds and skill levels.
- Keep criticism about the code, the data, or the idea - never the person.
- No harassment, personal attacks, discrimination, or unwelcome attention.
  This holds in issues, pull requests, commit messages, and any other
  project space.
- English isn't everyone's first language, and that's fine. A clumsy bug
  report is still a useful bug report.

## If something goes wrong

If someone is behaving badly, or you've been treated unfairly, contact the
maintainer privately through GitHub rather than escalating in a public
thread. Reports are handled in confidence and in good faith.

The maintainer may edit, hide, or remove comments, commits, issues, and
pull requests that don't follow this, and may block repeat offenders from
the project.

This is adapted in spirit from the
[Contributor Covenant](https://www.contributor-covenant.org); it is
deliberately short for a project this size, but the intent is the same.
